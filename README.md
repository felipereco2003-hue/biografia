## Resumen del Proyecto

El proyecto consiste en el desarrollo de una **página web personal** que presenta tu **biografía**, incluyendo información relevante sobre tu vida, formación y experiencias. Además, integra un **formulario de contacto** donde los usuarios pueden ingresar sus datos para comunicarse contigo de manera directa.  
El objetivo principal es ofrecer una plataforma **informativa, accesible y funcional**, que permita a los visitantes conocerte mejor y establecer contacto de forma sencilla.
